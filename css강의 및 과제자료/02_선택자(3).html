문자열 속성 선택자

태그에 설정된 속성, 속성값으로 태그를 찾아서 style을 적용

  1. = : 동일한 값을 찾을 때
  2. ^= : 시작하는 문자가 동일한 값을 찾을 때
  3. |= : -를 기준으로 동일한 값을 찾을 때
  4. $= : 끝나는 문자가 동일한 값을 찾을때
  5. *= : 포함된 값을 찾을 때
  6. ~= : 띄어쓰기를 기준으로 동일한 값을 찾을 때

속성값이 정확히 www인 태그나 www바로 다음에 하이픈으로 시작하는 요소


www naver
ww naver
naver www
wwwn aver
wwwn naver www naver
www-naver ww-naver naver-www wwwn-aver wwwn-naver-www-naver
