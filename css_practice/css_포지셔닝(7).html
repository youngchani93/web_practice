position 속성

태그를 페이지에서 원하는 위치에 배치할 때 사용하는 속성
기본적으로 태그는 순차적으로 배치를 한다.
static : 작성된 태그의 순서대로 배치 (default)
absolute : 부모의 position이 static일 때, 앞 페이지의 왼쪽 꼭지점이 0,0으로 기준
relative : 부모의 position이 static일 때, 부모의 왼쪽 꼭지점이 0,0으로 기준
*부모가 absolute나 relative일 때, 모두 부모의 왼쪽 꼭지점이 0,0 fixed : 태그의 위치를 특정 위치로 고정하는 속성

relative도 있다!!
